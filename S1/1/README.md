---

# Algorithum

```
step 1 : start

step 5 : stop

```

# Code

 

## c
### save using .c formalt
[> ***Full code*** <](https://github.com/Ajay-o-s/Collage-codes/blob/main/1/add.c)<br> 

```
#include<stdio.h> 
void main()
{
    int number1,number2,sum;
    printf("Enter Number1 : "); 
    scanf("%d",&number1); 
    printf("Enter Number2 : ");
    scanf("%d",&number2); 
    sum = number1 +number2;
    printf("SUM = %d \n",sum);
}

```
